Excel Formulas Practice Workbook

This repository contains a practical Excel workbook designed to help users understand and practice essential Excel formulas and functions.

The workbook walks through core spreadsheet concepts ranging from basic arithmetic formulas to logical and lookup functions used in real-world data analysis.

Workbook Content

The Excel file includes structured sheets covering the following topics:

- Formula Fundamentals
- Tax Rate Calculations
- Order of Operations in Excel
- Basic Functions
- Conditional Functions
- Text Functions
- Logical Functions
- Lookup Functions
- Date & Time Functions
- Bonus Exercises

Each sheet demonstrates practical examples that help users understand how Excel formulas work and how they can be applied to real datasets.

Skills Covered
This workbook helps learners practice:

- Basic arithmetic formulas
- SUM, AVERAGE and other core functions
- Conditional logic using IF statements
- Text manipulation functions
- Logical functions
- Lookup functions for retrieving data
- Date and time calculations

These skills are essential for:

- Data analysis
- Business reporting
- HR and administrative dashboards
- Financial calculations
- Spreadsheet automation

Who This Is For

This project is useful for:
- Beginners learning Excel formulas
- Students practicing spreadsheet skills
- Data analysts building strong Excel foundations
- HR and administrative professionals working with datasets
